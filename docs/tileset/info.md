# `info` - table
Provides wavewall high-level information about the tileset. At the moment, there's not much needed here, but stay tuned!

# Fields
## `size` - table
Gives the size in pixels of a tile in the tileset

### Fields
- `height`: number
- `width`: number

::: details Example
```lua
info = {
    size = {
        height = 32,
        width = 32,
    },
}
```
:::
